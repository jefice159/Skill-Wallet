CSS Box Model

The box model defines spacing and sizing of elements with margin, padding, border, width, and height.

Box 1: With Margin & Padding

Box Model Explained:

  • Content: The actual text or elements inside
  • Padding: Space between content and border (inside)
  • Border: Line around the box
  • Margin: Space outside the border (outside)

Flexbox Properties:

  • display: flex: Makes container a flex container
  • flex-direction: row (horizontal) or column (vertical)
  • justify-content: Aligns items horizontally (flex-start, center, space-between, space-around, flex-end)
  • align-items: Aligns items vertically (flex-start, center, flex-end, stretch)

Styled Forms

Forms are styled using padding, border-radius, background-color, and other CSS properties.

Form Styling Properties:

  • padding: Space inside input fields
  • border: Input field borders
  • border-radius: Rounded corners on buttons and inputs
  • background-color: Button and input background colors
  • font-size: Text size in form elements
  • color: Text color in form elements

Layout Best Practices:

  • Semantic HTML: Use header, main, footer, section tags
  • Flexbox Container: Use display: flex for layout control
  • Consistent Spacing: Use margin and padding uniformly
  • Visual Hierarchy: Use different font sizes and colors
  • Responsive Design: Layouts adapt to different screen sizes
